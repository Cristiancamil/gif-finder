## COMUNICACIÓN ENTRE COMPONENTES DESDE HIJO PADRE - PADRE HIJO
Para poder comunicar componentes padres a hijos y viseversa se puede hacer
pasando funciones como propiedad al componente hijo.

## Componente padres
export cont ComponentePadre = () => {
  const handleTermClicked = (term: string) => {
    console.log({term})
  }

  return (
    <>
      <ComponenteHijo onLabelClicked={handleTermClicked}/>
    </>
  )
}

# Componente hijo
interface Props {
  onLabelClicked: (term:string) => void
}

export const ComponenteHijo = ({onLabelClicked}: Props) {
  return (
    <>
      <li onClick={() => onLabelClicked(term)}></li>
    </>
  )
}


## Eventos comunes utilizados en React
Los eventos permiten ejecutar acciones cuando el usuario interactúa con la interfaz.

onClick(): Se ejecuta cuando el usuario hace clic sobre un elemento, se usa en botones, enlaces y tarjetas.
onChange(): Se ejecuta cuando cambia el valor de un elemento en el formulario, se usa en input, select y textarea.
onKeyDown(): Se ejecuta cuando el usuario presiona una tecla.
onKeyUp(): Se ejecuta cuando el usuario suela una tecla.
onSubmit(): Se ejecuta cuando se envía un formulario.
onMouseEnter(): Se ejecuta cuando el cursor entra sobre un componente.
onMouseLabel(): Se ejecuta cuando el cursor sale del elemento.
